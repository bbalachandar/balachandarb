Project

A smart shopping trolley for visionless people using Raspberry pi

This project helps the visually impaired people in collecting their groceries. >A shopping trolley is a necessary tool for shopping in supermarkets. In this project Raspberry pi is deployed in the robot module as trolley along with RFID reader, headset, motors. On the other hand, if the trolley is full means it difficult to carry by Blind people. So that motor is used. The voice of the person like section where the person wants to go is taken as input and this input is communicated via Bluetooth to Raspberry pi.
